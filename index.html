The hide and show code

A quick run-through

This is just a quick little script that hides all content on page load and then individually displays them as you click on the navigation list items. All content that you want to hide should belong to the hidables class. The section id must be section_ followed by a sequential number that matches the id number of the list item in the navigation, which looks like id="icn_[number]"

You can take a look at the show_hide.js file to see what's going on, and you also need Jquery.
